# Multi-Criteria Dijkstra Pathfinder

A JavaFX desktop application that finds the best route between two nodes using Dijkstra’s algorithm.

This project was built for a Design and Analysis of Algorithms course. The main idea is to solve a realistic shortest-path problem where the user can choose the best path based on distance, time, or both.

## Features

- Load a graph from a text file
- Find the shortest path based on distance
- Find the fastest path based on travel time
- Compare distance-based and time-based routes
- Use Dijkstra’s algorithm with a custom priority queue
- Display the final path and total cost through a JavaFX interface

## Technologies Used

- Java
- JavaFX
- Maven
- Dijkstra’s Algorithm
- Graphs
- Custom Priority Queue
- Custom Linked List / Array List
- File Handling
